DVineStructure.truncate

DVineStructure.truncate(self, trunc_lvl: int) → None

Truncates the R-vine structure.

While a structure of dimension d contains at most d-1 nested levels, this function extracts a sub-structure based on a given truncation level.

If the structure is already truncated at a level less than trunc_lvl, the function does nothing.

Parameters:

trunc_lvl – The truncation level.
